# FindIt - AI Personal Item Locator

## Project Overview

Design a mobile-first web application called FindIt.

FindIt helps users record, manage and search for important personal belongings.

The core idea is:

Users can save where an item is placed, upload photos, record location descriptions, and maintain historical location records.

When users forget where an item is, they can quickly search and find its latest location and historical records.

Future versions will include AI image recognition and natural language search.
